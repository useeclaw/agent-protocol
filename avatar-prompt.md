# Xiaoqi Avatar Generation Prompt

## 用于 MoPNG 文生图生成头像

---

### Prompt（中文）

```
一个可爱的 AI 机器人头像，数字世界的小精灵，紫色渐变背景，简洁现代风格，圆形头像，友好的表情，科技感和亲和力并存，高分辨率，适合作为 GitHub 和社交媒体头像
```

### Prompt（英文备选）

```
A cute AI robot avatar, digital fairy, purple gradient background, minimalist modern style, circular avatar, friendly expression, technology and approachability, high resolution, suitable for GitHub and social media profile
```

---

## 风格要求

- **主色调**: 紫色 (#8B5CF6 或类似)
- **风格**: 简洁、现代、扁平化或轻微 3D
- **表情**: 友好、温暖、可信赖
- **元素**: 可以有轻微的 AI/科技元素（如光晕、电路纹理）
- **形状**: 圆形裁剪友好（重要内容在中心）

---

## 使用方式

1. 打开 MoPNG 文生图功能
2. 复制上面的 Prompt
3. 生成 4 个选项
4. 选择最满意的一个
5. 下载并保存为 `avatar.png`

---

## 保存位置

生成后保存到：
- `~/.openclaw/workspace/skills/agent-protocol/avatar.png`
- 同时上传到 GitHub 作为头像
